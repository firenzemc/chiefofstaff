# Connectors (Commercial)

This directory is reserved for closed-source commercial connectors.

## Why Closed Source?

The core Mainframe framework (transcription, intent extraction, routing protocol) is open source under Apache 2.0. The connectors—which integrate with specific business systems—are our commercial differentiation.

## Included Connectors (Commercial License)

- ERP integrations (SAP, Oracle, local ERPs)
- WMS integrations (旺店通, 聚水潭)
- CRM integrations (Salesforce, 销售易)
- Industry-specific models

## Building Your Own Connector

If you'd like to build a custom connector, see our connector protocol specification in the main documentation.
